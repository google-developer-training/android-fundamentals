KeyboardSamples - Solution Code
===============================

App for experimenting with the android:inputType attribute for the EditText UI
element.

Introduction
------------

This app changes the input keyboard so that it suggests spelling corrections
and capitalizes each new sentence, and displays the entered text
in a toast message.

Pre-requisites
--------------

You need to know:
- How to open, build, and run apps with Android Studio.

Getting Started
---------------

1. Download the code.
2. Open the code in Android Studio.
3. Run the app.
